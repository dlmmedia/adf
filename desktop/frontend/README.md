# DLM ADF Frontend

The frontend application for DLM ADF by GenTech. Built with React 19, TypeScript, Vite, and Mantine UI.

## Development

```bash
npm install
npm run dev          # Start dev server at http://localhost:5173
npm run tauri-dev    # Start Tauri desktop app with hot reload
```

## Build

```bash
npm run build        # Web production build
npm run tauri-build  # Desktop installers (DMG/MSI/DEB/RPM)
```

## Testing

```bash
npm run test         # Run tests with vitest
npm run typecheck    # TypeScript type checking
npm run lint         # ESLint
npm run check        # All checks (typecheck + lint + test)
```
